# Hungrify 🍽️

Hungrify is a **MERN Stack web application integrated with Botpress AI** that helps users discover restaurants based on their **budget and number of people**.

This project was developed as part of the **Intro to Future Studies** course where we were asked to present a futuristic idea and implement it if possible.

## 💡 Idea

Instead of manually searching for restaurants, users interact with an **AI chatbot**.

The chatbot asks two questions:
- Number of persons
- Budget
- Location

Based on this information, the system suggests **restaurants within the user's budget** and also recommends better options if the budget is slightly increased.

## 🚀 Tech Stack

- MongoDB  
- Express.js  
- React.js  
- Node.js  
- Botpress AI (Chatbot)
