рџ“Њ Overview

Hungrify is a MERN stack web application integrated with Botpress AI that helps users discover restaurants based on their budget and number of people.

Instead of manually searching through multiple restaurants, users can simply interact with an AI chatbot which asks two questions and then suggests suitable restaurants nearby within the user's budget.

This project was developed as part of the Intro to Future Studies course project, where the goal was to propose and implement a futuristic idea.

рџ’Ў Idea

Finding restaurants within a specific budget for a group of people can be time-consuming.

Hungrify solves this problem by:

Asking the user the number of persons

Asking the budget

The AI assistant then:

Recommends restaurants within the given budget

Suggests better restaurant options if the budget is slightly increased
